.. _sessions:

登陆
----------------------

POST `/api/sessions`

post data
----------------------

================ ================================= ======================= =======================
key              meaning                           format                  comment
================ ================================= ======================= =======================
email            用户邮箱 
password         用户密码                       
================ ================================= ======================= =======================

response
--------------------

success
~~~~~~~
::

    {
        "status": 0,
        "message": "success",
        "data": {}
    }

fail
~~~~
::

    {
        "status": status_code,
        "message": "",
        "data": {}
    }
